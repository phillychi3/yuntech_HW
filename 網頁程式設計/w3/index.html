奶油義大利麵的食譜

材料表:

  • 義大利麵
  • 奶油
  • 蒜末
  • 鹽
  • 黑胡椒粉
  • 巴馬臣芝士

步驟:

  1. 將義大利麵煮熟，瀝乾備用。
  2. 在鍋中加熱奶油，加入蒜末炒香。
  3. 將瀝乾的義大利麵加入鍋中，拌炒均勻。
  4. 加入鹽和黑胡椒粉調味，繼續拌炒。
  5. 起鍋前灑上巴馬臣芝士，完成。

糖葫蘆的食譜

材料表:

  • 小番茄
  • 紅糖
  • 水
  • 竹籤

步驟:

  1. 將小番茄洗淨，插上竹籤。
  2. 將紅糖和水放入鍋中，煮至糖溶化。
  3. 將糖水煮至濃稠，關火。
  4. 將番茄籤快速地在糖水中滾一遍，放涼即成。
